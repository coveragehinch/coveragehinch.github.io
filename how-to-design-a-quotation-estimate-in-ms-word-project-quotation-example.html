A project quotation is a document for determining the cost and time required for the completion of any project. Usually, a quotation document prepared for a project is valid for a specific period. If work on the project cannot be started during this period, then a new project quotation is prepared by re-determining the cost and time of the project.

In this video, we have designed a similar project quotation template for you. By watching it from beginning to end you can learn how to design a quotation document yourself with a professional and excellent layout.
