This recipe from grandma stunned everyone! I've never eaten such a delicious cake! Unsurprisingly, This is one of my all-time favorite recipes. Creamy cream, delicate cakes. INGREDIENTS
VANILLA CREAM :
All-purpose flour ( 1 tbsp )
Milk ( 500 ml )
2 eggs
Sugar ( 135 g )
Cornstarch ( 30 g )
Butter ( 170 g )

CAKE PASTRY :
Milk ( 80 ml )
Water ( 80 ml )
Butter ( 60 g )
Sugar ( 2 tbsp )
Salt ( 1/2 tsp )
All-purpose flour ( 95 g )
3 eggs

2 SPRINGFORM PANS ( 20 CM )

BAKE AT 190 DEGREES
FOR 25-30 MINUTES
