In this video, I give a brief overview of the new XLOOKUP function in Excel, an alternative to VLOOKUP or INDEX MATCH.

Here are some of the major features of XLOOKUP that I share in the video:

1. Only 3 Required Arguments
2. Exact Match by Default
3. Separate Lookup & Return Ranges
4. Vertical OR Horizontal Lookup
5. Partial Match with Wildcards
6. Reverse Search Order
7. Cause of # VALUE! Error
8. Availability & Compatibility

The XLOOKUP function searches for the lookup value in a column/row and returns a value from an adjacent row/column where the match is found.

It has options for exact match, closest match, and partial match with wildcard characters. It can also search in reverse order.
