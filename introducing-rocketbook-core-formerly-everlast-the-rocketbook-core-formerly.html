The Rocketbook Core (formerly Everlast: https://bit.ly/372jXxn) notebook provides a classic pen and paper experience, yet is built for the digital age. Although it feels like a traditional notebook, the Core is endlessly reusable and connected to all of your favorite cloud services.

When you write using any pen from the Pilot Frixion line, your writing sticks to Core pages like regular paper. But add a drop of water… and the notebook erases like magic.
