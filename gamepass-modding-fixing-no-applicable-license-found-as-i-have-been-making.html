As I have been making these recent videos on modding gamepass games I keep having this same error pop up.
When I go from a modded state to unmodded state by uninstalling and reinstalling I often get an error.
The error states there are no applicable licenses found for my application.
This video walks you through how to use powershell to fix the problem

command
Get-AppXPackage | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
