Today we are going to discuss about different meaning and usage of LIKE and WOULD LIKE.

Like = ចូលចិត្តអ្វី១ ចូលចិត្តធ្វើអ្វី១ (ជាទូរទៅ)
I like + something.
I like + to do something
+ doing something.
Would like = (១) ចង់ ចង់បាន ចង់ញុំា ( want)
Would like + something.
Would like + to do something.
Would you like: (២)ប្រើសម្រាប់ការផ្តល់អ្វី១ (ពេលផ្តល់អ្វី១ដល់គេ)

Like = ចូលចិត្តអ្វី១ ចូលចិត្តធ្វើអ្វី១ (ជាទូរទៅ)
I like + something.
I like cat.
I like you.
I like snow.

I like + to do (something).
I like to read books.
I like to play football.

I like + doing (something)
I like reading books.
I like playing football.

I would like (I’d like)
I’d like (polite, diplomatic) , I want (demand )
I’d like + something
I’d like a cup of coffee.
I’d like sandwich.
I’d like some rice.

I’d like + to do (something)
I’d like to apply for a job.
I’d like to meet you.
I’d like to eat chicken.
I’d like to drink coke.
She’d like to buy a new car.

Would you like….?
Would you like something to drink?
Would you like some coffee?
Would you like some more tea?
Would you like more rice?
Would you like to join us?
