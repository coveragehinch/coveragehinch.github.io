To log on to this remote computer, you must be granted the Allow log on through Terminal Services right.
By default, members of the Remote Desktop Users group have this right. If you are not a member of the Remote Desktop
Users group or another group that has this right, or if the Remote Desktop Users group does not have this right, you must be
granted this right mannually..
