More information about the MagicPad™

*Kids can create art that glows with the all-new MagicPad™!
*MagicPad™ features an amazing crystal-clear, light-up screen that not only brings art to life, but also makes learning fun!

*MagicPad’s screen easily wipes clean and is portable for fun on the go
*It also has a built-in stand, so kids can display their creations, or it makes for a perfect nightlight.
*Each set includes a MagicPad™, 3 Dual-Sided Neon Pens (for six total colors), 30 Stencils, a Glow-Boost Card, Eraser Cloth and a Fun Guide.
